# Problème de Transport — Recherche Opérationnelle

## Description
Modélisation et résolution d'un problème de transport 
3 entrepôts / 4 magasins avec PuLP et GLPK.

## Modèle mathématique
- Variables : x_ij = quantité transportée de Ei vers Mj
- Objectif : minimiser le coût total de transport
- Contraintes : respect des capacités et des demandes

## Résultats
Coût optimal : 490

## Outils
- Python 3.12
- PuLP
- GLPK
